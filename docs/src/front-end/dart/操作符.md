| 描述           | 操作符                                                               |
| -------------- | -------------------------------------------------------------------- |
| 后缀操作       | `expr++` `expr--` `()` `[]` `?[]` `.` `?.` `!`                       |
| 前缀操作       | `-expr` `!expr` `~expr` `++expr` `--expr` `await expr`               |
| 乘除           | `*` `/` `%` `~/`                                                     |
| 加减           | `+` `-`                                                              |
| 位移           | `<<` `>>` `>>>`                                                      |
| 按位与         | `&`                                                                  |
| 按位异或       | `^`                                                                  |
| 按位或         | `\|`                                                                 |
| 关系和类型判断 | `>=` `>` `<=` `<` `as` `is` `is!`                                    |
| 相等           | `==` `!=`                                                            |
| 逻辑与         | `&&`                                                                 |
| 逻辑或         | `\|\|`                                                               |
| 是否为空       | `??`                                                                 |
| 三目运算       | `expr1 ? expr2 : expr3`                                              |
| 级联           | `..` `?..`                                                           |
| 赋值           | `=` `*=` `/=` `~/=` `%=` `+=` `-=` `<<=` `>>=` `&=` `^=` `\|=` `??=` |
